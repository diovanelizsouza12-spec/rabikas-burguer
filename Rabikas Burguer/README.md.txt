# Rabika's Burguer

Sistema de Cardápio Digital + Painel Administrativo

## Funcionalidades

- Cardápio digital completo
- Pedido via WhatsApp
- Delivery + Retirada
- Controle de horário automático
- Painel Administrativo
- Alteração de preços
- Criação de categorias
- Impressão manual de pedidos

## Horário de funcionamento

Terça a Domingo  
18:30 às 22:30

## Projeto

Desenvolvido para a lanchonete Rabika's Burguer.
